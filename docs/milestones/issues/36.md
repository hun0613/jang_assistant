## 스코프 정의

### 현재 화면에서 필요한 기능
| 화면 | 필요한 API | 설명 |
|------|-----------|------|
| `/createCart` | createCartItem | 품목 추가 (name, quantity) |
| `/createCart` | deleteCartItem | 품목 삭제 |
| `/createCart`, `/shopping/[id]` | getCartItemsByCartId | 장바구니별 품목 목록 조회 |
| `/shopping/[id]` | updateCartItem | 품목 수정 (quantity, price, status) |
