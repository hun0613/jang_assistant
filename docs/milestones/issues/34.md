## 현재 데이터 모델 분석

### localStorage 키 구조
| 키 | 용도 | 타입 |
|---|------|------|
| `title` | 장바구니 이름 | string |
| `memo` | 메모 | string |
| `cartItems` | 품목 목록 | CartItemType[] |
| `shoppingHistory` | 장보기 진행 상태 | CartItemType[] |

| `createCartGuideShown` | 가이드 노출 여부 | string (timestamp) |

### CartItemType
```typescript
type CartItemType = {
  id: number;
  name: string;
  quantity: number;
  price?: number;
  status: 'IN_CART' | 'IN_LIST';
};
```

## 스키마 설계

### 설계 원칙
- localStorage의 flat한 데이터 구조를 관계형 DB로 정규화
- 익명 세션 ID 기반으로 데이터를 식별 (Phase 2에서 user_id로 전환 예정)
- 가이드 노출 여부(`shoppingUsageGuideShown`, `createCartGuideShown`)는 UI 상태이므로 DB가 아닌 localStorage에 유지
- `carts.status` 컬럼 추가: Phase 3 장바구니 목록 관리 시 상태별 필터링 대비

### 테이블 구조

#### 1. `sessions` (익명 세션 관리)

| 컬럼 | 타입 | 제약조건 | 설명 |
|------|------|----------|------|
| `id` | uuid | PK, default: gen_random_uuid() | 세션 고유 ID |
| `created_at` | timestamptz | default: now() | 생성일시 |

- Phase 2 인증 도입 시 `user_id` 컬럼 추가하여 전환 가능

#### 2. `carts` (장바구니)

| 컬럼 | 타입 | 제약조건 | 설명 |
|------|------|----------|------|
| `id` | bigint | PK, auto increment | 장바구니 ID |
| `session_id` | uuid | FK → sessions.id, NOT NULL | 세션 ID |
| `title` | text | NOT NULL | 장바구니 이름 (최대 20자) |
| `memo` | text | | 메모 (최대 100자) |
| `status` | text | NOT NULL, default: 'CREATED' | 장바구니 상태 |
| `created_at` | timestamptz | default: now() | 생성일시 |
| `updated_at` | timestamptz | default: now() | 수정일시 |

**status 값:**
| 값 | 설명 |
|----|------|
| `CREATED` | 장바구니 생성 완료 |
| `SHOPPING` | 장보기 진행 중 |
| `COMPLETED` | 장보기 완료 |

#### 3. `cart_items` (품목)

| 컬럼 | 타입 | 제약조건 | 설명 |
|------|------|----------|------|
| `id` | bigint | PK, auto increment | 품목 ID |
| `cart_id` | bigint | FK → carts.id, NOT NULL | 장바구니 ID |
| `name` | text | NOT NULL | 품명 |
| `quantity` | integer | NOT NULL, default: 1 | 수량 |
| `price` | integer | | 단가 (장보기 시 입력) |
| `status` | text | NOT NULL, default: 'IN_LIST' | 상태 |
| `created_at` | timestamptz | default: now() | 생성일시 |

**status 값:**
| 값 | 설명 |
|----|------|
| `IN_LIST` | 리스트에 있음 (아직 카트에 안 담음) |
| `IN_CART` | 카트에 담음 |

### 테이블 관계도

```
sessions (1) ──→ (N) carts (1) ──→ (N) cart_items
```

### localStorage 유지 항목 (DB 저장 안 함)
| 키 | 사유 |
|----|------|
| `shoppingUsageGuideShown` | UI 상태 (기기별 1회 노출 제어) |
| `createCartGuideShown` | UI 상태 (기기별 1회 노출 제어) |
