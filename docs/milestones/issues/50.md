# #50 Supabase 리전 이전 (ap-northeast-2 → us-east-1)

## 이슈 정보

| 항목 | 내용 |
|------|------|
| 마일스톤 | Phase 1.5 - UX 보완 (로딩/에러 처리) |
| 목표 | Vercel iad1과 같은 리전으로 Supabase 이전하여 네트워크 레이턴시 해소 |
| 브랜치 | `refactor/#50` |

## 배경

| 항목 | Before | After |
|------|--------|-------|
| Vercel Function | iad1 (미국 버지니아) | iad1 (변경 없음) |
| Supabase DB | ap-northeast-2 (서울) | **us-east-1 (미국 버지니아)** |
| Vercel↔DB RTT | ~200ms (태평양 왕복) | **~10ms (같은 리전)** |
| 전체 응답시간 (예상) | ~800ms | **~420ms (50% 개선)** |
