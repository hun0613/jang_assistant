# 장비서 (Jang-Assistant) 정식 버전 로드맵

기획안의 IA, 요구사항 정의서, MVP 피드백을 기반으로 정리한 로드맵이다.

## 개발 일정

### 완료 (2025)

| 기간 | 항목 | 상태 |
|------|------|------|
| 2025.06 | MVP 개발 | 완료 |
| 2025.07 | MVP 배포 및 피드백 수집 | 완료 |
| 2025.07~08 | 피드백 반영 및 방향성 수정 | 완료 |

### 정식 버전 (2026~)

| 기간 | 항목 | Phase |
|------|------|-------|
| 2026.03~ | 정식버전 요구사항 개발 | Phase 1~3 |
| TBD | 정식버전 QA | - |
| TBD | 정식버전 운영배포 및 안정화 | - |
| TBD | 스프린트 진행 (서비스 고도화) | Phase 4~5 |

> 빠른 MVP 공개를 통해 시장 반응과 사용자 피드백을 수집 → 피드백을 토대로 기능 및 방향성 수정 → 정식버전 개발 진행 후 서비스 완성도를 높히기 위한 QA 진행

## Phase 1 - 백엔드 도입 및 데이터 영속화

| 항목 | 설명 |
|------|------|
| 목표 | localStorage 의존성 탈피, 데이터 영속 저장 |
| 영향 | 기기 간 데이터 동기화, 데이터 유실 방지 |

### 주요 작업
- **DB 기반 데이터 관리**: 장바구니, 품목, 장보기 내역 등 서버 저장
- **서버 API 구축**: CRUD API 설계 및 구현

---

## Phase 2 - 소셜 로그인 및 사용자 기능

| 항목 | 설명 |
|------|------|
| 목표 | 사용자 인증 및 개인화 기능 도입 |
| 영향 | 사용자별 데이터 분리, 개인 설정 관리 |

### 주요 작업
- **소셜 로그인**: Google, Kakao, Naver OAuth 연동
- **마이페이지**: 닉네임 변경, 회원 탈퇴, 로그아웃
- **홈 페이지**: 로그인 후 진입하는 메인 허브 페이지

---

## Phase 3 - 장바구니 목록 관리

| 항목 | 설명 |
|------|------|
| 목표 | 여러 개의 장바구니를 테마별로 생성/관리 |
| 영향 | "캠핑 장보기", "명절 준비" 등 상황별 리스트 관리 가능 |

### 주요 작업
- **장바구니 목록 페이지**: 생성한 장바구니 리스트 조회
- **장바구니 CRUD**: 생성, 조회, 수정, 삭제
- **장바구니 상세페이지**: 품목 확인, 장보기 시작, 수정 진입
- **장바구니 수정페이지**: 품목 편집, 이름/메모 수정
- **장보기 UX 개선** (MVP 피드백 반영):
  - 품목 정렬: '담지않은 품목' 상단, '담은 품목' 하단 배치
  - 심플모드: [품목 추가 → 체크] 만으로 사용 가능한 간소화 모드
  - 담기 UI 개선: 모달 대신 리스트에서 바로 체크할 수 있도록 변경

---

## Phase 4 - 지출 관리 및 데이터 시각화

| 항목 | 설명 |
|------|------|
| 목표 | 장보기 지출 내역 기록 및 소비 패턴 분석 |
| 영향 | 가계부 기능, 계획적 소비 지원 |

### 주요 작업
- **장보기 내역**: 전체/월별 조회, 장보기내역 상세페이지
- **지출 내역 대시보드**:
  - 이달 지출 현황
  - 최근 6개월 간 지출 현황 차트
- **정보입력 페이지**: 장보기 완료 시 결제 정보 기록
- **영수증 이미지 첨부** (선택)

---

## Phase 5 - 부가 기능 및 서비스 고도화

| 항목 | 설명 |
|------|------|
| 목표 | 사용자 경험 향상 및 타겟 확장 |
| 타겟 확장 | 계획형 소비자 → 즉흥적인 소비를 즐기는 사람들까지 |

### 후보 기능
- **장바구니 공유 서비스**: 생성한 장바구니를 커뮤니티 기능을 통해 다른 사람들에게 공유
- **AI 레시피 재료 자동 생성**: 요리 레시피에 필요한 재료들을 AI가 자동 생성하여 장바구니로 제공
- **테마별 AI 추천 장보기 리스트**: 장보기 목적(자취생 식료품 구매, MT, 캠핑)을 토대로 AI가 필요한 물품을 자동 생성
- **과거 구매 이력 검색**
- **자주 사는 품목 추천**
