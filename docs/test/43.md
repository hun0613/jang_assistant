# #43 커스텀 404 페이지 구현 - TC

## TC-1: 글로벌 404 페이지 동작 확인

**절차:**
1. 존재하지 않는 경로에 접근한다 (예: `/asdfgh`)

**기대 결과:**
- [ ] 로고 이미지가 표시된다
- [ ] "유효하지 않은 페이지 입니다!" 문구가 표시된다
- [ ] "홈으로 돌아가기" 버튼이 표시된다
- [ ] 버튼 클릭 시 홈(`/`)으로 이동한다

---

## TC-2: 장바구니 상세 404 페이지 동작 확인

**절차:**
1. 존재하지 않는 장바구니 ID로 접근한다 (예: `/shopping/999999`)

**기대 결과:**
- [ ] 로고 이미지가 표시된다
- [ ] "유효하지 않은 페이지 입니다!" 문구가 표시된다
- [ ] "홈으로 돌아가기" 버튼이 표시된다
- [ ] 버튼 클릭 시 홈(`/`)으로 이동한다

---

## TC-3: 잘못된 장바구니 ID 형식 처리

**절차:**
1. 문자열 ID로 접근한다 (예: `/shopping/abc`)

**기대 결과:**
- [ ] 404 페이지가 표시된다 (숫자가 아닌 ID는 notFound 처리)

---

## TC-4: 데이터 패칭 속도 개선 확인

**절차:**
1. 정상적인 장바구니 상세 페이지에 접근한다 (예: `/shopping/1`)

**기대 결과:**
- [ ] 페이지 로딩 시 깜빡임 없이 데이터가 바로 표시된다 (SSR)
- [ ] 장바구니 제목, 품목 목록이 정상적으로 표시된다
